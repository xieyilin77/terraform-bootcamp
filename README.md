# terraform-bootcamp

## Project Overview

This repository contains the hands-on exercises completed during the Terraform Bootcamp. The project demonstrates Infrastructure as Code (IaC) concepts using Terraform to provision and manage cloud infrastructure.

## Objectives

- Learn Terraform fundamentals
- Write reusable Terraform configurations
- Manage infrastructure using Terraform CLI
- Understand Terraform state management
- Use variables, outputs, and modules
- Deploy AWS resources using Terraform
- Follow Infrastructure as Code best practices
